# templates/vault/

Starter vault templates used by the /setup wizard's "Create new vault" path. They mirror the shape Smithers expects:

```
<vault>/
├── Daily Notes/
├── Drafts/
│   ├── Originals/
│   └── Archived/
├── Call Notes/
├── Agendas/
├── Projects/
│   └── <slug>/
│       ├── info.md
│       ├── notes.md
│       └── agenda.md
├── Weekly Updates/
├── Working With You.md
├── <Your Name> Style Guide.md
├── Follow-ups.md
└── Templates/
    ├── Daily Note Template.md
    ├── Weekly Update Template.md
    ├── Project Template/
    └── Call Note Template.md
```

Files to be filled in as the vault layer lands.
